## âš™ Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ak-AhmedFathi/Next-Dashboard.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Next-Dashboard
   ```
3. Install dependencies:
   ```bash
   npm install
   ```


